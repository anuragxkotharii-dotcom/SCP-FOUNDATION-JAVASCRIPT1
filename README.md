# 📁 SCP Foundation JavaScript Project

A fan-made **SCP Foundation-style web application** built using HTML, CSS, and JavaScript. This project recreates the aesthetic and feel of SCP database entries while showcasing a custom SCP concept.

## 🧠 About the Project

This project is a **front-end web development experiment** inspired by the SCP Foundation universe. It simulates an SCP article interface and presents a custom SCP-001 entry titled **“Hero”**.

The goal of this project is to:

* Practice JavaScript-based UI behavior
* Recreate a recognizable fictional database design
* Experiment with storytelling through web interfaces

---

## ✨ Features

* 📄 SCP-style document layout
* 🎨 Themed UI inspired by SCP database design
* ⚙️ Interactive elements powered by JavaScript
* 🧩 Custom SCP-001 concept with detailed description
* 🌐 Fully deployed using GitHub Pages
  
---

## 🛠️ Tech Stack

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript** – Interactivity and dynamic behavior
* **GitHub Pages** – Deployment

## 📖 SCP-001: "Hero" (Fan Concept)

This project includes a **non-canon SCP-001 entry**, depicting an extremely powerful entity with abilities such as:

* Reality manipulation
* Infinite strength and intelligence
* Temporal control
* Guaranteed victory in all scenarios
